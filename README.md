# Tic Tac Toe

A terminal-based two-player Tic Tac Toe game written in Python.

## How to play

Two players share the keyboard and take turns placing `X` and `O` on a 3×3 grid.
Cells are numbered 1–9, laid out row by row:

```
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```

The first player to line up three of their marks in a row, column, or diagonal wins.
If the board fills with no winner, the game is a draw.

## Running the game

```bash
python tictactoe.py
```

Type `q` at any prompt to quit. After each game you can choose to play again.
